Mini Spotify XML

Mini Spotify XML es una aplicación web basada en Python (Flask) para gestionar una biblioteca musical, utilizando XML para almacenamiento y XSLT para la capa de presentación.

## Installation

Usa el gestor de paquetes pip para instalar las dependencias requeridas:


```
pip install -r requirements.txt
```


## Usage

Para iniciar el servidor de desarrollo ejecuta:

```
flask run
```
Una vez iniciado, abre tu navegador en:  

```
http://127.0.0.1:5000
```

Desde el dashboard podrás:

- **Crear:** Agregar canciones, artistas y álbumes automáticamente.  
- **Buscar:** Filtrar tu biblioteca en tiempo real.  
- **Gestionar:** Editar o eliminar elementos con integridad referencial (eliminación en cascada).
